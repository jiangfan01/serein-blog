# 动画规则

## 动画技术栈

- 主动画库：GSAP、Framer Motion
- 简单状态过渡可以使用 CSS transition

## 分工原则

- Framer Motion：组件入场、布局过渡、presence 切换
- GSAP：复杂时间线、滚动驱动动画、需要强控制力的编排动画

## 动效方向

- 动效要服务于层级、节奏和视觉聚焦
- 风格偏高级、克制、有意图，不做噪音式小动画堆砌
- 宁可少而强，也不要多而散

## 约束

- 动画不能影响内容可读性
- 重动画场景要考虑低性能设备的退化表现
- 不要默认给所有元素都加动画
- 重复出现的动画模式要抽成 hook、helper 或可复用封装
