# JavaScript 笔记

JavaScript 是一门动态、弱类型的编程语言，广泛用于 Web 开发。

## 核心概念

- 数据类型 - 基本类型与引用类型
- 原型链 - JavaScript 的继承机制
- 闭包 - 函数与词法作用域
- 异步编程 - Promise、async/await
- 事件循环 - 宏任务与微任务
