# CSS 笔记

CSS（层叠样式表）用于控制网页的视觉呈现。

## 核心概念

- 盒模型 - margin、border、padding、content
- 布局方式 - Flex、Grid、浮动、定位
- 选择器 - 基础选择器、组合选择器、伪类伪元素
- 响应式设计 - 媒体查询、视口单位
