# 斐波那契数列 - 动态规划

## 题目描述

给定一个正整数 `n`，请用动态规划的方法求出斐波那契数列的第 `n` 项。

斐波那契数列定义如下：
- `F(0) = 0`
- `F(1) = 1`
- `F(n) = F(n-1) + F(n-2)` （n ≥ 2）

### 示例

```js
fib(5); // 输出 5
fib(10); // 输出 55
```

## 解法：自底向上动态规划

```js
function fib(n) {
  if (n === 0) return 0;
  if (n === 1) return 1;

  const dp = [0, 1];
  for (let i = 2; i <= n; i++) {
    dp[i] = dp[i - 1] + dp[i - 2];
  }

  return dp[n];
}
```

## 思路解析

- 使用 `dp[i]` 表示第 `i` 个斐波那契数
- 初始条件为 `dp[0] = 0`，`dp[1] = 1`
- 从 `2` 开始迭代，使用前两项推出当前项
- 这是典型的"重叠子问题"且"最优子结构"问题，适合用动态规划解决
- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`（如需优化可改为常量空间）
