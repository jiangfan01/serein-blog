# React 笔记

React 是一个用于构建用户界面的 JavaScript 库，由 Facebook 开发和维护。

## 核心概念

- **JSX** - JavaScript 的语法扩展
- **组件** - 可复用的 UI 单元
- **Props & State** - 数据流管理
- **Hooks** - 函数组件的状态和副作用
- **Virtual DOM** - 高效的 DOM 更新机制
