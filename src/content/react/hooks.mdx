# React Hooks 详解

Hooks 是 React 16.8 引入的特性，让函数组件也能使用状态和其他 React 特性。

## useState - 状态管理

用于声明组件的状态变量，状态变化会触发组件重新渲染。

```tsx
const [count, setCount] = useState(0);

// 更新状态
setCount(count + 1);

// 函数式更新（推荐）
setCount(prev => prev + 1);
```

## useEffect - 副作用处理

用于处理副作用（如：数据获取、事件监听、订阅、DOM 操作等）。

```tsx
// 每次渲染都执行
useEffect(() => {
  console.log('渲染了');
});

// 只在挂载时执行一次
useEffect(() => {
  fetchData();
}, []);

// 依赖变化时执行
useEffect(() => {
  fetchUser(userId);
}, [userId]);

// 清理副作用
useEffect(() => {
  const subscription = subscribe();
  return () => {
    subscription.unsubscribe();
  };
}, []);
```

## useMemo - 缓存计算结果

用于缓存计算结果，当依赖不变时，避免执行重复或昂贵的运算。

```tsx
const expensiveValue = useMemo(() => {
  return heavyCalculation(data);
}, [data]);
```

## useCallback - 缓存函数引用

用于缓存函数引用，常用于将函数作为 props 传递给子组件。

```tsx
const handleClick = useCallback(() => {
  console.log("clicked");
}, []);

// 带依赖
const handleSubmit = useCallback(() => {
  submitForm(formData);
}, [formData]);
```

## useRef - 可变引用

用于创建可变的引用对象，常用于访问 DOM 或保存组件内的可变值。

```tsx
// 访问 DOM
const inputRef = useRef<HTMLInputElement>(null);

useEffect(() => {
  inputRef.current?.focus();
}, []);

return <input ref={inputRef} />;

// 保存可变值（不触发重渲染）
const countRef = useRef(0);
countRef.current += 1;
```

## useContext - 上下文访问

用于访问 React Context 中的数据，实现跨组件层级的状态共享。

```tsx
// 创建 Context
const ThemeContext = createContext('light');

// 提供 Context
<ThemeContext.Provider value="dark">
  <App />
</ThemeContext.Provider>

// 消费 Context
const theme = useContext(ThemeContext);
```

## useLayoutEffect - 同步副作用

与 useEffect 类似，但在 DOM 更新之后同步执行。适合需要读取 DOM 布局的场景。

```tsx
useLayoutEffect(() => {
  const rect = ref.current.getBoundingClientRect();
  // 同步处理布局
}, []);
```

> **建议**：优先使用 `useEffect`，仅在需要同步 DOM 读取时使用 `useLayoutEffect`。

## useReducer - 复杂状态管理

适用于复杂的状态逻辑，类似 Redux 的 reducer 模式。

```tsx
const initialState = { count: 0 };

function reducer(state, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      return state;
  }
}

const [state, dispatch] = useReducer(reducer, initialState);

// 使用
dispatch({ type: 'increment' });
```

## Hooks 使用规则

1. **只在顶层调用** - 不要在循环、条件或嵌套函数中调用
2. **只在 React 函数中调用** - 函数组件或自定义 Hook 中

```tsx
// ❌ 错误
if (condition) {
  const [state, setState] = useState();
}

// ✅ 正确
const [state, setState] = useState();
if (condition) {
  // 使用 state
}
```

## 总结

| Hook | 用途 |
|------|------|
| useState | 状态管理 |
| useEffect | 副作用处理 |
| useMemo | 缓存计算结果 |
| useCallback | 缓存函数引用 |
| useRef | DOM 引用 / 可变值 |
| useContext | 上下文访问 |
| useReducer | 复杂状态管理 |
| useLayoutEffect | 同步副作用 |
