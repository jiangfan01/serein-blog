## 🏗️ 架构概览

Workflow 工作流系统是一个基于 **React Flow** 的可视化流程编辑器，支持：
- 拖拽式节点编辑
- 串行/并行执行流程
- 图形化配置智能体工作流
- 实时执行状态监控
- 撤销/重做操作

### 核心架构层级

```
用户界面层 (WorkFlowEdit.tsx)
    ↓
React Flow 可视化层 (ReactFlowCanvas.tsx)
    ↓
节点组件层 (WorkflowReactFlowNode.tsx + 各类 Module)
    ↓
状态管理层 (workflowStore.ts + flowHistoryStore.ts)
    ↓
数据转换层 (workflowConfigConverter.ts)
    ↓
API 层 (workflowApi.ts)
    ↓
后端服务 (agent_config JSON)
```

## 🔄 图转树与树转图

这是整个系统的核心转换逻辑，位于 `workflowConfigConverter.ts`。

### 图 → 树：buildAgentConfig（提交时）

**使用场景**：用户编辑完工作流，点击保存/提交时

**核心算法**：DFS（深度优先搜索）遍历图结构

```typescript
export const buildAgentConfig = (
  nodes: WorkflowNode[],
  edges: Edge[],
  convertNode: NodeConverter
): AgentConfig => {
  // 1. 构建邻接表和入度表
  const outEdges: Record<string, Edge[]> = {};
  const inDegree: Record<string, number> = {};
  
  edges.forEach(edge => {
    outEdges[edge.source] = outEdges[edge.source] || [];
    outEdges[edge.source].push(edge);
    inDegree[edge.target] = (inDegree[edge.target] || 0) + 1;
  });

  // 2. 找根节点（入度为0）
  const rootNodes = nodes.filter(node => !inDegree[node.id]);

  // 3. DFS 构建分支
  const buildBranch = (startNodeId: string): (AgentNode | Stage)[] => {
    const result = [];
    let currentId = startNodeId;

    while (currentId && !processed.has(currentId)) {
      const currentNode = nodes.find(n => n.id === currentId);
      const nodeOutEdges = outEdges[currentId] || [];

      if (nodeOutEdges.length > 1) {
        // 并行分支：创建 parallel stage
        result.push(convertNode(currentNode));
        result.push({
          parallel: true,
          branches: nodeOutEdges.map((edge, index) => ({
            id: `branch${index}`,
            nodes: buildBranch(edge.target)
          }))
        });
        break;
      } else if (nodeOutEdges.length === 1) {
        // 串行：继续遍历
        result.push(convertNode(currentNode));
        currentId = nodeOutEdges[0].target;
      } else {
        // 叶子节点
        result.push(convertNode(currentNode));
        break;
      }
    }
    return result;
  };

  // 4. 从根节点开始构建
  const stages = [];
  rootNodes.forEach(root => {
    const branchResult = buildBranch(root.id);
    stages.push(...branchResult);
  });

  return { type: "stage", stages };
};
```

**转换示例**：

```
图结构：
  A → B → C
      ↓
      D

树结构：
{
  type: "stage",
  stages: [
    { parallel: false, nodes: [A] },
    { parallel: false, nodes: [B] },
    { 
      parallel: true, 
      branches: [
        { id: "branch1", nodes: [C] },
        { id: "branch2", nodes: [D] }
      ]
    }
  ]
}
```

### 树 → 图：parseAgentConfig（回填时）

**使用场景**：编辑已有工作流，从后端加载数据时

**核心算法**：DFS 遍历树结构，生成节点和边

**布局常量**：
- `VERTICAL_SPACING = 450` - 垂直间距
- `HORIZONTAL_SPACING = 700` - 水平间距（并行分支）

```typescript
export const parseAgentConfig = (
  configStr: string,
  processedStages?: ProcessedStage[]
): ParseResult => {
  const config: AgentConfig = JSON.parse(configStr);
  const nodes: Node[] = [];
  const edges: Edge[] = [];
  let sequence = 1;

  // 创建 React Flow 节点
  const createReactFlowNode = (agentNode: AgentNode, position) => {
    const nodeId = `node-${agentNode.agent_id}-${sequence}`;
    return {
      id: nodeId,
      type: "workflowNode",
      position,
      data: {
        id: nodeId,
        sequence: sequence++,
        agent_id: agentNode.agent_id,
        agent_name: agentNode.output_key,
        params: agentNode.message_template.metadata,
        user_template: agentNode.message_template.user_template
      }
    };
  };

  // DFS 遍历分支
  const traverseBranch = (
    branchNodes: (AgentNode | Stage)[],
    startY: number,
    offsetX: number,
    parentNodeIds: string[]
  ) => {
    let currentY = startY;
    let prevNodeIds = parentNodeIds;

    for (const item of branchNodes) {
      if ("parallel" in item && item.branches) {
        // 并行阶段：横向展开
        const branchCount = item.branches.length;
        const startX = offsetX - (branchCount - 1) * HORIZONTAL_SPACING / 2;

        item.branches.forEach((branch, index) => {
          const branchX = startX + index * HORIZONTAL_SPACING;
          traverseBranch(branch.nodes, currentY, branchX, prevNodeIds);
        });
      } else if ("agent_id" in item) {
        // 普通节点：纵向排列
        const node = createReactFlowNode(item, { x: offsetX, y: currentY });
        nodes.push(node);

        // 创建边
        prevNodeIds.forEach(prevId => {
          edges.push({
            id: `edge-${prevId}-${node.id}`,
            source: prevId,
            target: node.id,
            type: "customEdge"
          });
        });

        currentY += VERTICAL_SPACING;
        prevNodeIds = [node.id];
      }
    }
  };

  // 处理顶层 stages
  config.stages.forEach(stage => {
    if (stage.parallel && stage.branches) {
      stage.branches.forEach((branch, index) => {
        traverseBranch(branch.nodes, currentY, branchX, lastNodeIds);
      });
    } else if (stage.nodes) {
      stage.nodes.forEach(agentNode => {
        const node = createReactFlowNode(agentNode, { x: 0, y: currentY });
        nodes.push(node);
        currentY += VERTICAL_SPACING;
      });
    }
  });

  return { nodes, edges };
};
```

## 🎨 React Flow 集成

### ReactFlowCanvas 组件

核心画布组件，封装了 React Flow 的所有功能。

**关键特性**：
1. **自动布局**：使用 Dagre 算法自动计算节点位置
2. **API 暴露**：通过 `onReactFlowReady` 回调暴露操作方法
3. **历史记录**：集成撤销/重做功能
4. **响应式**：支持主题切换、移动端适配

```typescript
const ReactFlowCanvasInner: React.FC<ReactFlowCanvasProps> = ({
  initialData,
  nodeTypes,
  edgeTypes,
  onReactFlowReady,
  onNodesChange
}) => {
  // 1. 使用 React Flow Hooks
  const [nodes, setNodes, onNodesChangeInternal] = useNodesState([]);
  const [edges, setEdges, onEdgesChange] = useEdgesState([]);
  const { fitView, getNodes, getEdges } = useReactFlow();

  // 2. Dagre 自动布局
  const getLayoutedElements = (nodes, edges) => {
    const dagreGraph = new dagre.graphlib.Graph();
    dagreGraph.setGraph({ rankdir: "TB", nodesep: 100, ranksep: 200 });
    
    nodes.forEach(node => 
      dagreGraph.setNode(node.id, { width: 600, height: 300 })
    );
    edges.forEach(edge => 
      dagreGraph.setEdge(edge.source, edge.target)
    );
    
    dagre.layout(dagreGraph);
    
    return nodes.map(node => {
      const positioned = dagreGraph.node(node.id);
      return {
        ...node,
        position: { x: positioned.x - 300, y: positioned.y - 150 }
      };
    });
  };

  // 3. 初始化布局
  useEffect(() => {
    if (initialData?.nodes.length > 0) {
      const layoutedNodes = getLayoutedElements(initialData.nodes, initialData.edges);
      setNodes(layoutedNodes);
      setEdges(initialData.edges);
      setTimeout(() => fitView({ padding: 0.2, duration: 300 }), 100);
    }
  }, [initialData]);

  // 4. 暴露 API 给父组件
  useEffect(() => {
    if (onReactFlowReady) {
      onReactFlowReady({
        getNodes: () => apiRef.current.getNodes(),
        getEdges: () => apiRef.current.getEdges(),
        addNode: (node) => apiRef.current.addNode(node),
        removeNode: (nodeId) => apiRef.current.removeNode(nodeId),
        undo: () => apiRef.current.undo(),
        redo: () => apiRef.current.redo()
      });
    }
  }, []);

  return (
    <ReactFlow
      nodes={nodes}
      edges={edges}
      onNodesChange={onNodesChange}
      onEdgesChange={onEdgesChange}
      nodeTypes={nodeTypes}
      edgeTypes={edgeTypes}
      proOptions={{ hideAttribution: true }}
    >
      <Background variant={BackgroundVariant.Dots} />
      <Controls />
      <MiniMap />
    </ReactFlow>
  );
};
```

### 2. flowHistoryStore - 撤销/重做

管理画布操作历史，支持撤销和重做。

**核心机制**：
```typescript
interface FlowHistoryState {
  undoStack: Snapshot[];         // 撤销栈
  redoStack: Snapshot[];         // 重做栈
  initialized: boolean;          // 是否已初始化
  maxHistory: number;            // 最大历史记录数（50）
}

interface Snapshot {
  nodes: Node[];
  edges: Edge[];
}
```

**操作流程**：

1. **记录快照**：每次节点/边变化时调用 `pushSnapshot`
2. **撤销**：从 `undoStack` 弹出上一个状态，当前状态推入 `redoStack`
3. **重做**：从 `redoStack` 弹出下一个状态，当前状态推入 `undoStack`
4. **新操作**：任何新操作都会清空 `redoStack`

**快捷键**：
- `Cmd/Ctrl + Z` - 撤销
- `Cmd/Ctrl + Shift + Z` - 重做

```typescript
const handleRemoveNode = (nodeId: string) => {
  // 1. 删除业务数据
  workflowStore.removeNode(nodeId);
  
  // 2. 删除可视化节点和相关边
  setNodes(nodes.filter(n => n.id !== nodeId));
  setEdges(edges.filter(e => e.source !== nodeId && e.target !== nodeId));
  
  // 3. 重新计算序列号
  const reorderedNodes = nodes.map((node, index) => ({
    ...node,
    data: { ...node.data, sequence: index + 1 }
  }));
  
  // 4. 记录历史
  flowHistoryStore.pushSnapshot({ nodes: reorderedNodes, edges });
};
```

```typescript
const handleUpdateNode = (nodeId: string, newData: Partial<WorkflowNode>) => {
```

### 边操作

#### 1. 连接规则

```typescript
// React Flow 连接验证
const isValidConnection = (connection: Connection) => {
  const { source, target } = connection;
  
  // 规则1：不能连接自己
  if (source === target) return false;
  
  // 规则2：不能形成环路
  if (hasCircle(source, target, edges)) return false;
  
  // 规则3：目标节点只能有一个输入（单入度）
  const targetHasInput = edges.some(e => e.target === target);
  if (targetHasInput) return false;
  
  return true;
};
```

#### 2. 创建连接

```typescript
const onConnect = (connection: Connection) => {
  if (!isValidConnection(connection)) return;
  
  const newEdge: Edge = {
    id: `edge-${connection.source}-${connection.target}`,
    source: connection.source,
    sourceHandle: "bottom",
    target: connection.target,
    targetHandle: "top",
    type: "customEdge",
    animated: false
  };
  
  setEdges([...edges, newEdge]);
  flowHistoryStore.pushSnapshot({ nodes, edges: [...edges, newEdge] });
};
```

#### 3. 删除连接

```typescript
const onEdgesDelete = (deletedEdges: Edge[]) => {
  const remainingEdges = edges.filter(
    edge => !deletedEdges.some(de => de.id === edge.id)
  );
  
  setEdges(remainingEdges);
  flowHistoryStore.pushSnapshot({ nodes, edges: remainingEdges });
};
```

## 🔌 Hooks 使用

### 1. useSubmitWorkflow

负责工作流的提交逻辑。

```typescript
const useSubmitWorkflow = ({ editId, cronExpression, getReactFlowData }) => {
  const [isSubmitting, setIsSubmitting] = useState(false);
  const workflowStore = useWorkflowStore();
  
  const submitWorkflow = async () => {
    setIsSubmitting(true);
    
    try {
      // 1. 获取 React Flow 数据
      const reactFlowApi = getReactFlowData();
      const nodes = reactFlowApi.getNodes();
      const edges = reactFlowApi.getEdges();
      
      // 2. 提取业务数据
      const workflowNodes = nodes.map(node => node.data);
      
      // 3. 转换为后端格式
      const agentConfig = buildAgentConfig(
        workflowNodes,
        edges,
        (node) => ({
          agent_id: node.agent_id,
          output_key: node.agent_name,
          message_template: {
            user_template: node.user_template,
            metadata: node.params
          }
        })
      );
      
      // 4. 构建提交数据
      const submitData = {
        name: workflowStore.workflowName,
        description: workflowStore.workflowDescription,
        agent_config: JSON.stringify(agentConfig),
        cron_expression: workflowStore.isScheduled ? cronExpression : []
      };
      
      // 5. API 调用
      if (editId) {
        await updateWorkflow(editId, submitData);
      } else {
        await createWorkflow(submitData);
      }
      
      // 6. 成功后跳转

    } catch (error) {
      console.error("提交失败:", error);
    } finally {
      setIsSubmitting(false);
    }
  };
  
  return { submitWorkflow, isSubmitting };
};
```

```typescript
const useWorkflowStatus = (): UseWorkflowStatusReturn => {
  const [workflowStatus, setWorkflowStatus] = useState<WorkflowStatus | null>(null);
  const [isConnected, setIsConnected] = useState(false);
  const [error, setError] = useState<string | null>(null);

    const unsubscribeMessage = taskWebSocketChannel.onMessage((rawData) => {
      // 处理单节点状态消息
      if (rawData.action === "workflow_status_change") {
        const message = rawData as WorkflowStatusMessage;
        
        setWorkflowStatus({
          taskId: message.task_id,
          workId: message.work_id,
          status: message.status,
          agent_name: message.current_node?.node_name || "执行中",
          timestamp: message.timestamp,
          currentNode: message.current_node
        });
        
        // 执行完成后清空状态
        if (message.status === "success" || message.status === "failed") {
          setTimeout(() => setWorkflowStatus(null), 3000);
        }
      }
      
      // 处理分支执行消息
      if (rawData.action === "branches_executing") {
        const message = rawData as BranchesExecutingMessage;
        
        // 计算整体状态
        const allSuccess = message.branches.every(b => b.status === "success");
        const hasFailed = message.branches.some(b => b.status === "failed");
        const overallStatus = allSuccess ? "success" : hasFailed ? "failed" : "executing";
        
        setWorkflowStatus({
          taskId: message.task_id,
          workId: message.work_id,
          status: overallStatus,
          agent_name: message.branches
            .filter(b => b.status === "executing")
            .map(b => b.current_node?.node_name || b.branch_id)
            .join(", "),
          timestamp: message.timestamp,
          branches: message.branches
        });
      }
    });
    
    // 2. 订阅连接状态
    const unsubscribeConnection = taskWebSocketChannel.onConnectionChange((connected) => {
      setIsConnected(connected);
    });
    
    // 3. 连接 WebSocket
    if (!taskWebSocketChannel.isConnected()) {
      taskWebSocketChannel.connect();
    }
    
    return () => {
      unsubscribeMessage();
      unsubscribeConnection();
    };
  }, []);
  
  return { workflowStatus, isConnected, error };
};
```

### 5. 错误处理

```typescript
// 提交前验证
const validateWorkflow = () => {
  // 1. 检查工作流名称
  if (!workflowName.trim()) {
    throw new Error("请输入工作流名称");
  }
  
  // 2. 检查节点数量
  if (nodes.length === 0) {
    throw new Error("请至少添加一个节点");
  }
  
  // 3. 检查孤立节点
  const isolatedNodes = nodes.filter(node => {
    const hasInput = edges.some(e => e.target === node.id);
    const hasOutput = edges.some(e => e.source === node.id);
    return !hasInput && !hasOutput && nodes.length > 1;
  });
  
  if (isolatedNodes.length > 0) {
    throw new Error("存在未连接的节点");
  }
  
  // 4. 检查环路
  if (hasCircle(nodes, edges)) {
    throw new Error("工作流不能包含环路");
  }
};
```
