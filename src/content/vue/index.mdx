# Vue 笔记

Vue 是一个用于构建用户界面的渐进式 JavaScript 框架。

## 核心概念

- 响应式系统 - Vue 3 使用 Proxy 实现
- 组合式 API - setup、ref、reactive
- 模板语法 - 指令、插值、事件
- 组件通信 - props、emit、provide/inject
- 生命周期 - 组件从创建到销毁的钩子
